基底 · paper-50 护眼暖纸

全局背景色定于 paper-50 #FBF7F0 — 温暖的米纸色，绝非纯白。所有色阶、墨色、强调色均以此为基础协调搭配。

为什么选择 paper-50？

纯白 #FFFFFF 在长时间阅读时容易导致视觉疲劳，而 paper-50 的暖色调（RGB 251, 247, 240）减少了蓝光刺激， 模拟真实纸张的微妙暖意。琥珀主色 #C67E2E 与 paper-50 形成经典的"金纸合一"搭配， 墨蓝辅色 #4A7BB5 提供沉稳对比，所有色彩在这个基底上都显得柔和、自然、有呼吸感。

响应式设计 · 三断点

≥768px / ≤768px / ≤480px 三断点覆盖，双栏优先，移动端回退单列。

断点宽度布局策略
≥768px平板+桌面双栏 Grid / 多栏自适应 / 纸边 20px
≤768px大手机单列堆叠 / 纸边 16px / 字号 ≥ 14px
≤480px小手机紧凑间距 / 纸边 12px / 标题降级

移动端必配清单 · 8 项硬性检查

  • <meta viewport> — width=device-width, initial-scale=1
  • 三断点覆盖 — ≥768 / ≤768 / ≤480
  • 字号 ≥ 14px — 移动端全文最小
  • 触控 ≥ 44px — 所有交互元素最小区域
  • 无水平溢出 — max-width:100% + overflow-x:hidden
  • 图片自适应 — max-width:100% + height:auto
  • 布局降级 — 双栏 → 单列
  • Safe Area — env(safe-area-inset-*)
/* ── 移动端适配最低模板 ── */ *, *::before, *::after { box-sizing: border-box; } html { scroll-behavior: smooth; } img { max-width: 100%; height: auto; } .grid-2 { display: grid; grid-template-columns: repeat(2, 1fr); gap: var(--sp-lg); } @media (max-width: 768px) { .grid-2 { grid-template-columns: 1fr; } body { padding: 0 var(--sp-md); } } @media (max-width: 480px) { body { padding: 0 var(--sp-sm); } h1 { font-size: var(--fs-title1); } }
