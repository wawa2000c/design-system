图标库 · 核心框架

基于「纸本美学 · 编辑风格」设计规范的线稿图标体系。viewBox 24×24，描边 1.8，圆角端点，琥珀主色 / 墨蓝辅色。本期为框架版，含 8 大类共 64 枚，确认风格后扩充至 200。

核心框架 v1 · 64 / 200 · 作者 wawa2000c

图标几何

stroke-width 1.8
stroke-linecap / linejoin round
fill none · stroke currentColor
禁用 emoji，全手绘线稿 SVG
尺寸梯度

使用准则

✓ 推荐

  • 统一 1.8 描边
  • 圆角端点
  • 24 栅格对齐
  • currentColor 继承文字色

✕ 避免

  • 混用实心/线稿
  • 描边忽粗忽细
  • 使用 emoji
  • 随意改 viewBox
